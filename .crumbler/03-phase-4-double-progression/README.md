# Phase 4: Double Progression

Implement double progression for accessory work.

## Features to Implement

- **Progression: `DoubleProgression`** - Add reps until hitting ceiling, then add weight and reset reps
- **SetScheme: `RepRange(min, max)`** - Flexible rep targets within a range

## Programs Unlocked

| Program | Why |
|---------|-----|
| Reddit PPL 6-Day | Double progression for accessories, failure handling for main lifts |

## Acceptance Criteria

- Users can define rep ranges for sets
- System tracks when user hits top of rep range
- Weight increases automatically when rep ceiling is reached
- E2E tests demonstrate Reddit PPL 6-Day program
